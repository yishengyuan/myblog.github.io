# ✅ **一、国际通用英语等级（CEFR）**

CEFR（欧洲共同语言参考标准）是全球最常用的英语能力分级体系，从 A1 到 C2。

| CEFR等级     | 能力描述                 | 对应水平          |
| ---------- | -------------------- | ------------- |
| **A1 初级**  | 能进行非常简单交流（自我介绍、日常词汇） | 小学/零基础        |
| **A2 初中级** | 能处理简单任务（购物、问路等）      | 初中水平          |
| **B1 中级**  | 能在熟悉话题上表达观点          | 高中/四级边缘       |
| **B2 中高级** | 能熟练交流，阅读技术文章         | 大学四级~六级       |
| **C1 高级**  | 能在专业环境熟练表达           | 六级优秀/雅思7+     |
| **C2 母语级** | 近母语水平                | 雅思8.5+ / 专八优秀 |

# ✅ **四、工作岗位常用英语等级（企业内部）**

一些外企或大厂会用业务能力来划分：

| 企业分类          | 英语要求描述       | 对应CEFR |
| ------------- | ------------ | ------ |
| **0级：无需英语**   | 内部中文交流即可     | A1–A2  |
| **1级：基本读写**   | 能看懂简单文档      | B1     |
| **2级：能邮件沟通**  | 可与海外团队邮件往来   | B1–B2  |
| **3级：工作交流**   | 能开会、电话、做技术讨论 | B2     |
| **4级：独立对接客户** | 能主持会议、谈判     | C1     |
| **5级：专业表达**   | 近母语级，能演讲     | C2     |
