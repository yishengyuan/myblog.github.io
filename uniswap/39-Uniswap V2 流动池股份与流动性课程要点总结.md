### Uniswap V2 流动池股份与流动性课程要点总结

1. **流动池股份计算公式**
   - 流动池股份的计算公式为：
      份额量 = (L1 - L0) / L0 * 总份额 T
   - 其中，L1 是添加流动性后池子的价值，L0 是添加流动性前池子的价值。
2. **L1 - L0 / L0 与 DX / X0 和 DY / Y0 的关系**
   - 公式 L1 - L0 / L0 可以等于 DX / X0 或 DY / Y0。
   - 这表明：
     - DX / X0 表示添加的 Token X 与池中原有 Token X 的比值。
     - DY / Y0 表示添加的 Token Y 与池中原有 Token Y 的比值。
3. **流动池价值函数的三种定义**
   - 有三种方法可以定义流动池的价值函数：
     1. f(X, Y) = √(X * Y)（几何平均数）
     2. f(X, Y) = 2 * X
     3. f(X, Y) = 2 * Y
   - 在这三种定义下，公式 L1 - L0 / L0 = DX / X0 或 DY / Y0 都是成立的。
4. **股份计算公式的替换**
   - 使用上述的关系式替换 L1 - L0 / L0 后，可以得到股份计算公式：
     - 份额量 = DX / X0 * 总份额 T
     - 也可以表示为：份额量 = DY / Y0 * 总份额 T。
5. **总结**
   - 流动池股份量的计算与添加的 Token 数量与池中原有 Token 数量的比值成正比。
   - 不同的流动池价值函数定义方式最终得出的股份计算公式是相同的。
