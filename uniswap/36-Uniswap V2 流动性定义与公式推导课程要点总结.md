### Uniswap V2 流动性定义与公式推导课程要点总结

1. **引入流动性定义**

   - 在上一节课中提到，L1 - L0 / L0 等于 DX / x0，并且等于 DY / y0。我们尚未推导出这个公式，并且还没有明确 L0 和 L1 的定义。
   - L1 是指添加了 DX 和 DY 数量的 Token 后的流动性，L0 是指添加这些 Token 前的流动性。

2. **流动性定义的三种方式**

   - 我们可以通过三种方式来定义流动性函数，这样公式才能成立。

3. **流动性函数的定义**

   - 函数 f

     ：用来衡量流动池的价值，接受两个输入（Token X 和 Token Y 的数量），输出一个数值 L。

     - **第一种定义方式**：f(x, y) = √(x * y)，即 X 和 Y 的几何平均数。
     - **第二种定义方式**：f(x, y) = 2 * X，即流动性等于 X 的两倍。
     - **第三种定义方式**：f(x, y) = 2 * Y，即流动性等于 Y 的两倍。

4. **流动性与公式的关系**

   - 当流动性按照这三种方式定义时，上述的公式 L1 - L0 / L0 = DX / x0 和 L1 - L0 / L0 = DY / y0 都会成立。

5. **总结**

   - 本节课介绍了如何定义流动性，并通过三种不同的方式来衡量流动池的价值。
