## Uniswap V2 课程导学要点

### 课程源代码

https://github.com/POKEDT/uni-v2-code

### 课程介绍

欢迎加入 Uniswap V2 课程。在本视频中，我们将回答以下几个问题：
1. 这门课程适合哪些人？
2. 为什么要学习这门课程？
3. 你将在课程中学习什么内容？
4. 如何有效学习这门课程？

### 1. 课程对象
- 本课程适合中级 Solidity 开发者，特别是对 DeFi 感兴趣并有 Foundry 使用经验的开发者。
- 具备基础的高中数学知识会有帮助，但不是必须的。

### 2. 为什么要学习这门课程？
- Uniswap V2 是许多协议的基础，因此深入了解 Uniswap V2 的工作原理会为您在安全审计或漏洞赏金比赛中提供竞争优势。
- 通过学习 Uniswap V2，您将掌握构建其他自动化做市商（AMM）的背景知识，有助于扩展到其他 AMM 协议，比如 Curve 和 Uniswap V3。

### 3. 课程内容
- **常数乘积 AMM 的数学原理**：深入理解 AMM 的数学原理，这是 Uniswap V2 合约运作的核心。
- **Uniswap V2 合约深入剖析**：我们会分析合约中的代码逻辑和功能，并进行练习，帮助你熟练掌握合约的使用。
- **练习与模拟交易**：您将有机会在主网模拟交易（无需支付交易费用），实际操作 Uniswap V2 合约，提高动手能力。

### 4. 有效学习的最佳实践
- **动手操作**：观看视频后，建议在 Foundry 和主网 Fork 上多练习，模拟交易以加深对课程内容的理解。
- **构建项目**：尝试编写应用程序，集成 Uniswap V2 合约，或复现 Uniswap V2 合约代码，这样可以帮助您理解合约设计的原因。
- **做笔记**：将笔记保存至 GitHub 私有仓库，便于日后查阅，尤其是在需要回顾协议内容时。
- **无需死记硬背**：无需记住公式或函数调用方法，通过做笔记和操作，关键内容会自然记住，不重要的部分随时可以通过笔记找回。
