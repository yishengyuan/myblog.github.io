## css后处理器

post-css

在css样式生成后做一些自动化处理，添加额外的功能

[https://postcss.org/](https://postcss.org/)

```jsx
module.exports = {
  plugins: {
    tailwindcss: {},
    autoprefixer: {},
  },
}
```

## tailwind

通过class来定义css样式的框架，使用非常方便

[https://tailwindcss.com/docs/installation](https://tailwindcss.com/docs/installation)

```jsx
module.exports = {
  content: [
    "./src/**/*.{js,jsx,ts,tsx}"
  ],
  theme: {
    extend: {},
  },
  plugins: [],
}
```
