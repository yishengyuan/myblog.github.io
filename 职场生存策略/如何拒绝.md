## **① 高情商拒绝加活（不抗拒、不硬顶、有逻辑）**

**目标：不当“免费劳力”，但保留专业形象与合作态度**

| 场景        | 可直接说的话                                  |
| --------- | --------------------------------------- |
| 工作满负荷，想婉拒 | **为了保证质量，我需要先处理好A和B，您看C要不要延后或者找其他人支援？** |
| 临时加急任务    | **我可以接，但需要明确优先级，我怕同时推进影响整体交付效果。**       |
| 想拒绝但不撕破脸  | **这个任务我能做，但如果由X来执行可能效率更高，我可以提供支持。**     |
| 明显消耗型任务   | **我理解这件事很紧急，但是否能换成我更擅长的方向，会更快出结果。**     |

**关键词：不说“不能做”，改成“怎样做最好”。**

## **② 高情商争取资源（不给资源=不做白工）**

**你的付出要和回报绑定。做得越多，权力越大。**

| 场景      | 可直接说的话                                 |
| ------- | -------------------------------------- |
| 争取机会/权限 | **这个任务我想负责，我需要XX权限/预算/数据才能推进得更快更稳。**   |
| 想接更核心项目 | **我对A项目有一些优化方案，如果能参与，我相信能提升整体效率和结果。**  |
| 干活但资源紧张 | **目前资源不足会影响交付，我建议增加X或放掉Y任务，您觉得哪种更合适？** |
| 希望被信任主导 | **这个模块我能独立负责，您可以把最终结果交给我来保证。**         |

**底层逻辑：换资源，而不是义务劳动。**
## **③ 高情商提升存在感（不拍马屁但让领导离不开你）**

**存在感不是汇报次数，是汇报方式。**

|场景|可直接说的话|
|---|---|
|主动汇报进度|**目前A已完成xx%，遇到B风险但我已有xxx预案，预计xx时间完成。**|
|让领导知道你能扛事|**这件事我已提前考虑风险并规划了应对方案，交给我放心。**|
|做出成果时|**这个结果达成的关键点是我们提前规划了X和Y，执行落地顺利。**|
|想表现价值|**这次成果证明我们走的方向是对的，后续我会进一步推进xx指标。**|

**要点：少讲过程，多讲结果和可复制的价值。**

## **④ 高情商建立信任（稳、靠谱、能托底）**

**领导用的人不是最累的人，是最稳的人。**

|场景|可直接说的话|
|---|---|
|任务接下时|**我先梳理清楚整体推进路径，关键节点会提前跟您同步。**|
|需要汇报关键决策|**我提供两个方案：A风险低但周期长；B效率高但资源需求多，您倾向哪种？**|
|出错或延期时|**问题已定位，我给出三个修复方案并评估了影响范围，不会再次发生。**|
|长期建立信任|**您交给我的事，我一定给结果；给不了结果，我提前给预案。**|

**核心不是讨好，而是让领导觉得“你用起来安心”。**

---

### 最后，我把四类话术浓缩成一句总准则：

> **能做的说明路径，难做的提条件，不做的给方案；  
> 给结果前做汇报，拿成果时说价值。**
