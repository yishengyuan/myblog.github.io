## 如何查看一个项目应该使用哪个node版本和hardhat版本

1. 检查package.json 
```js
"engines": {
  "node": ">=16.0.0 <18.0.0"
}

```
2. 查看 `dependencies` 和 `devDependencies` 

```js
"devDependencies": {
  "hardhat": "^2.6.0"
}

```

3. 查看 `hardhat.config.js` 或 `hardhat.config.ts` 文件
```js
module.exports = {
  solidity: "0.8.4",
  networks: {
    hardhat: {
      chainId: 1337
    },
  },
};

```

public BigDecimal getBalance(String address) throws IOException {  
// 获取价格
    EthGetBalance getBalance = web3j.ethGetBalance(address, DefaultBlockParameterName.LATEST).send();  
    return Convert.fromWei(getBalance.getBalance().toString(), Convert.Unit.ETHER);  
}
