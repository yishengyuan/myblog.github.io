## 深度的概念

深度反映了市场在不同价格水平上的订单数量，是流动性的重要指标。

  

  

深度计算逻辑

深度信息通过订单簿聚合计算：

java

// 买单深度：从最高价开始累计

 TreeMap<BigDecimal,MergeOrder> buyQueue = buyLimitPriceQueue; 

// 卖单深度：从最低价开始累计

 TreeMap<BigDecimal,MergeOrder> sellQueue = sellLimitPriceQueue;

深度特点：

- ·价格层级：每个价格水平的总订单量
- ·累计深度：从最优价格到当前价格的累计订单量
- ·实时更新：随订单变化动态调整
