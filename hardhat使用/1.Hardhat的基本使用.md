## 环境安装

---

# 更新 Homebrew（可选）

brew update

  

# 安装 yarn（会自动安装 node 作为依赖，或者如果你已有 node，这也可以）

brew install yarn

  

# 验证

yarn --version

## 第一个合约

---

```bash
mkdir firstcontract

cd firstcontract

npm install -g yarn

yarn init -y

yarn add -D hardhat

code .    // 打开vscode


// 管理员模式在power shell中执行
Set-ExecutionPolicy -Scope Process -ExecutionPolicy Bypass


npx hardhat --init

选择2.0   typescript

查看nodejs版本 
node --version
查看hardhat版本

```
