部署
安装 hexo-deployer-git
npm install hexo-deployer-git –save
修改配置 ：
npm list hexo-deployer-git
在config.yml文件下修改
deploy:
type: git
repo: https://github.com/yishengyuan/blog.git
branch: test

安装hexo-admin
npm install hexo-admin –save

在config.yml 中 增加
#hexo-admin 账号密码cookie
admin:
username: admin
password_hash: 123456
secret: 123456

使用阿里云绑定hexo博客github
在阿里购买域名 konglingfu.top
进入阿里云控制台
https://dc.console.aliyun.com/next/index#/domain/list/all-domain
进入解析设置
添加记录
记录类型: CNAME
记录值: yishengyuan.github.io
主机记录： www

记录类型: A
记录值: 108.160.165.147
主机记录： @

在git目录下创建一个CNAME文件 并将域名加入到其中
https://github.com/yishengyuan/blog/blob/test/CNAME
